Diamond Tech Website

This repository contains the source code for the Diamond Tech corporate website. The site is a responsive, single-page layout built with HTML5, Bootstrap 5, and minimal custom CSS. It is designed for clarity, performance, and easy deployment on Netlify or any static hosting provider.

Overview

The website presents Diamond Tech as a professional ICT solutions provider operating in Zimbabwe and South Africa. It includes the following sections:

Home (Hero section)

About Us

Products

Contact

Footer

All sections are accessible via anchor links in the navigation bar.

Technology Stack

HTML5

Bootstrap 5 (CDN)

Custom CSS

Open-source images (Unsplash)

No build tools, frameworks, or backend services are required.

Project Structure
/
├── index.html
└── README.md


This is a static site. All logic, styling, and content are contained within index.html.

Features

Fully responsive layout (mobile, tablet, desktop)

Bootstrap-based navigation bar and footer

Clean, corporate design suitable for ICT businesses

Open-source imagery (no licensing restrictions)

Smooth scrolling between sections

Easy to extend or convert to React/Vue in future
